# Roadmap

## P0 Core parity

- Rebuild core schema, controllers, and Google-like contact form layout.
- Implement dynamic add/remove rows for emails/phones/addresses/websites/events.
- Add labels and rich metadata support.
- Ensure robust list/search/filter UX.
- Introduce RSpec suite and coverage gates.

## P1 Advanced parity

- Photo upload + avatar display pipeline.
- Merge/dedupe workflow with preview and explicit conflict handling.
- Bulk actions and stronger search ranking.
- Import/export parity improvements.

## P2 Maximum parity

- Frequent/recent ranking heuristics.
- Suggested merge intelligence.
- Extended custom-field UX parity and advanced keyboard flows.
- Additional parity gaps identified through user feedback.
